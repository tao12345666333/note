# 操作快捷键(emacs 模式)

* Ctrl + a: 光标返回首位
* Ctrl + e: 光标移至末尾
* Ctrl + p: 上一个命令
* Ctrl + n: 下一个命令
* Ctrl + l: 清屏
* Ctrl + h: 回退一位
* Ctrl + b: 光标向左一位
* Ctrl + f: 光标向右一位
* Ctrl + u: 剪切光标前的内容(全部)
* Ctrl + w: 剪切光标前的内容(按词)
* Ctrl + k: 剪切光标后的内容
* Ctrl + y: 将剪切的内容复制到光标后
* Ctrl + t: 交换光标前的两个字符顺序
