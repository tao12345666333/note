# Git Start

## Introduction

分布式版本控制工具

## Setup and Config

```
git config --global user.name 'TaoBeier'
git config --global user.email 'zhangjintao9020@gmail.com'
```

## Repo

Create one local repo

```
git init
```

Create bare repo

```
git init --bare
```

## Methods

```
git add
git commit -m 'msg'
git checkout
...
```

## Branch and Merge

```
git branch

git merge
```

## Data transfer

```
git remote 
git push
git pull
git fetch
```

## Compare


```
git log

git diff 
```
